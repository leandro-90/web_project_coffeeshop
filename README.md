# Cafeteria Triple Espresso

Equanto você espera pelo seu café, a página oferece vídeos receitas onde você aprender e fazer na sua casa. Você pode reservar o seu café e outras goloseimas o dia e a hora que quiser. Não se esqueça de colocar seu nome e email para receber nossas promoções diárias.

A página foi construída usando as tecnologias...

- HTML5
- CSS3
- Metodologia de organização BEM e BEM FLat.
